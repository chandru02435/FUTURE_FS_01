🚀 Chandru A — Personal Portfolio Website
  A futuristic, AI-inspired personal portfolio website built with pure HTML, CSS, and JavaScript. Features animated particle systems, moving grid backgrounds, smooth scroll animations, and a fully responsive design.

🌐 Live Demo
👉 https://chandru02435.github.io/FUTURE_FS_01

✨ Features

  * Animated Background — Moving cyan grid and 150 floating star particles built with Canvas API
    Typing Animation — Auto-types and deletes role titles (Software Developer, UI/UX Designer, etc.)
    Scroll Reveal — Sections fade in smoothly as you scroll down
    Glowing Orbs — Three drifting background orbs for ambient depth
    Responsive Design — Works on all screen sizes (mobile, tablet, desktop)
    Contact Form — Functional form powered by Formspree with email notifications
    Hover Effects — Cards lift and glow on hover
    Glassmorphism Navbar — Sticky navbar with blur effect

   HTML
   CSS
   JAVA SCRIPT
   CANVAS API
